# KL2PEN Job Seeking Website
## Tech Stacks
1. Firebase
2. Gemini
3. React + Next.js
4. Tailwind CSS
5. shadcn/ui


# Getting Started
1. Clone the repository
2. Install dependencies
   ```bash
   npm install
   ```
3. Clone the .env.sample to .env
4. Insert your Firebase and Gemini API Token in the .env file
5. Run the app
   ```bash
   npm run dev
   ```
